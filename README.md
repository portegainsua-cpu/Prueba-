# Mi Laboratorio de Automatización e IA 🚀

¡Hola! En este repositorio voy a ir subiendo los planos (blueprints) de las automatizaciones No-Code que desarrollo en mi curso.

## Proyecto 1: Clasificador de Correos con IA
* **Herramienta usada:** Make 
* **Problema que soluciona:** Las empresas pierden mucho tiempo leyendo correos repetitivos.
* **Cómo funciona:** 1. Recibe el correo del cliente.
  2. Un modelo de lenguaje (GPT) evalúa si es una queja urgente o una duda común.
  3. Si es urgente, avisa al equipo por Slack/WhatsApp.
